## 🎮 项目概览

**项目名称**: Vue Tetris Game  
**项目描述**: 基于Vue3和HTML5 Canvas的经典俄罗斯方块游戏  
**技术栈**: Vue3 + Canvas + Pinia + Vite  
**开发模式**: 个人项目

### 🎯 核心特性
- 🧩 **经典玩法** - 传统俄罗斯方块游戏机制
- 🎨 **现代技术** - Vue3 Composition API + HTML5 Canvas
- 📱 **响应式设计** - 支持桌面和移动端
- 🎵 **音效系统** - 背景音乐和音效支持
- 💾 **数据持久化** - 最高分记录保存
